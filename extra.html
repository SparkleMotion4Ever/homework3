WURFL Device Capabilities

Below are three capabilities offered by the WURFL database that developers can use to test against the device requesting the application:

Capability Description
brand_name Returns the brand of the device, such as "Apple", "Samsung", or "Google". This allows developers to customize content or functionality based on the device's manufacturer.
model_name Provides the model name of the device, like "iPhone 12", "Galaxy S21", or "Pixel 6". This can be used to tailor the user experience for specific device models.
is_mobile Indicates whether the device is a mobile device. Returns true for mobile devices and false for desktops. This helps in serving mobile-optimized content or layouts.
